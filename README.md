# Algoritmos de Ordenação em C

Este repositório contém a implementação em C dos principais algoritmos de ordenação:

- Bubble Sort  
- Selection Sort  
- Insertion Sort  
- Quick Sort  
- Merge Sort  
- Shell Sort  

## Observações

- O programa gera um array aleatório e mostra o resultado antes e depois da ordenação.  
- Mede o tempo de execução de cada algoritmo para comparação de desempenho.
